Accessible Linear Stencil Optimization
Improving Access to State-Of-The-Art Computational Methods
Stencil Computation Model
The stencil model is used to represent problems which consist of an n-dimensional matrix of data whose cells change over time according to their neighbors as dictated by a stencil. [1]

This model is often used in the fields of Partial Differential Equations; (for example, heat transfer, and hydrodynamics). [1]
Introduction of FFT Method
  • Complexity of best stencil algorithms are linear in both grid size and time [1]
  • New stencil algorithm utilizes the FFT to allow for repeated squaring [1]
  • This algorithm is sublinear in either time or grid size [1]
Project Overview
  • When new computing methods are created, implementation options are needed
  • Pluto polyhedral optimizer does this through library [2]
  • Taco Compiler uses online code generation [3]
Fourier Stencil Computation Implementation Methods
This research project yielded two methods for code generation of the novel FFT-based method for periodic stencils.
FourSt Precompiler
gregory.croisdale.us/stencil/precompiler

FourSt is a C++ precompilation tool which takes C/C++ source code from stdin and returns an FFT-optimized stencil source to stdout.
FourSt Web Interface
gregory.croisdale.us/stencil/web

This is a static webpage which can take either a stencil kernel, coefficient matrix, or sparse matrix representation of the model and returns C++ code.
Future Work
  • Implement Aperiodic Stencils
    Currently, the code generators only generate code for periodic stencils. Substantial adjustments must be made in order to allow for boundary conditions.
  • Adapt FFT Method for non-linear stencils
    Research is now being done on creating the theoretical model for non-linear stencils optimized using the FFT. Upon completion, the code generators should be updated to reflect this improvement.
  • Create C/C++ Library
    A C/C++ Library, while not as transparent as a code generator, would simplify the installation and integration of the FFT-Optimized.
  • Create Python Library
    In order to be the most accessible to the researchers who would benefit from this optimization, a Python library should be created to seamlessly integrate with existing repositories and notebooks.
  • Automate benchmarking
    An easy and automated way to determine the best stencil computation method would be useful for those wishing to optimize their stencil code.
